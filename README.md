📘 Interactive Periodic Table of Elements

An interactive periodic table that helps users explore chemical elements in a visual and engaging way.

🚀 Features
Interactive periodic table UI
Click on elements to view detailed information
Displays:
Common compounds
Typical chemical reactions
Atomic structure models
Element classification (metal, nonmetal, noble gas, etc.)
